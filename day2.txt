What is Data,Field,Record,Database
-----------------------------------
What is Data ?
--------------
Raw Facts or figures without context.
Ex: 101,Ravi,50000,hyd

What is Field?
---------------
Smallest unit of data in a database(column/attribute)
Ex: Emp Name, EmpID, Emp Salary, Emp Address 

What is Record?
---------------
Collection of related fields(row/tuple)
Ex: (111,Ravi,60000,vjy)

What is database?
-----------------
Organised Collection of related records stored together.
Ex: An employee database containing all employee records

Client - Server Architecture 
----------------------------
Client - The role of the client in client server architecture is to send request to the server and to get response from the server
Server - Server is a software, the role of the server in client server architecture is to take request from the client. If the 
	 Request is valid request then it will generate some response we can handover that response to client. If client request
	 is not a valid request it will raise the errors
Protocol - It has the set of rules and regulations, its role is to send requests from client to server

Data Models in DBMS
-------------------
Def : Data Model defines how data is stored ,organised ,and manipulated in a database

1. Physical Data Model
=====================
Def : Describes how data is Physically stored in storage devices

2. Logical Data Model
====================
Defines the structure of data (entities, attributes, relationships ) independent of physical storage areas

Audience: Developers and Data Architects

3, Hierarchial Data Model
========================
Data is represented as Tree Structure (Parent-Child Relationship )
Ex: Company--> Department --> Employees

4. Network Data Model
====================
Data is represented as graph with records (nodes) and relationships (edges).

Ex : Student enrolled in multiple courses, courses taught by multiple professors

5. Relational Data Models(RDBMS)
================================
Data is stored in Tables(relations)with rows(tuples)and columns(attributes)

Ex : 
----
student(id, name, marks, age, dept_id)
department(dept_id, dept_name)

6. Entity Relationship (E-R Model)
==================================
A conceptual Model That shows entities(things),attributes(properties) and relationships.
